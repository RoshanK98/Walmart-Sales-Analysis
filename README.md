This project demonstrates an end-to-end Walmart Sales ETL Pipeline and Analytics Dashboard focused mainly on the Data Engineering workflow. The project involves extracting raw Walmart sales data from a CSV dataset, transforming and cleaning the data using Python, and loading the processed data into a PostgreSQL database. Various SQL queries were used to perform sales analysis and generate business insights such as total sales, top-performing stores, monthly sales trends, and holiday sales performance. Finally, an interactive Power BI dashboard was created to visualize the data using KPI cards, charts, and slicers. This project helped strengthen my practical skills in Python, ETL processes, PostgreSQL, SQL analysis, and Power BI while following a real-world Data Engineering workflow.
